# AI-Powered Resume Analyzer & Job Matcher

## Problem Statement
Recruiters receive thousands of resumes for each role. Traditional ATS systems
filter candidates using keyword matching, often rejecting qualified profiles.

## Solution
This project builds an AI-powered system that analyzes resumes and job
descriptions, intelligently matches candidates to roles, and explains skill
gaps and improvement areas.

## Key Features
- Resume parsing and skill extraction
- Job description understanding
- Intelligent match scoring
- Explainable AI feedback
- Skill gap analysis and recommendations

## Tech Stack
- Python
- NLP (spaCy, TF-IDF)
- Flask / FastAPI
- HTML, CSS, JavaScript

## Architecture
Resume → NLP Processing → Skill Normalization → Matching Engine → AI Insights → UI
